JavaScript

node.js é uma plataforma de desenvolvimento que permite executar JavaScript no lado do servidor.

typeof serve para verificar o tipo da variável

window é um objeto global que representa a janela do navegador

Para realizar conversão de dados de uma variável

para números usa-se:

  • Number(n) - número
  • Number.parseInt(n) - número inteiro
  • Number.parseFloat(n) - número real
  • String(n) - cadeia
  • n.toString() - cadeia
  • Boolean(n) - booleano
  • n.toFixed(2) - número com duas casas decimais
  • n.toPrecision(3) - número com três casas decimais
  • n.toExponential(2) - número em notação científica

para strings usa-se:

  • n.length - retorna o tamanho da string
  • n[0] - retorna o primeiro caractere da string
  • n.toUpperCase() - transforma em maiúsculas
  • n.toLowerCase() - transforma em minúsculas
  • n.charAt(0) - retorna o primeiro caractere
  • n.indexOf('a') - retorna a posição do caractere 'a'
  • n.replace('a', 'b') - substitui 'a' por 'b'
  • n.split(',') - divide a string em um array usando vírgula como separador
  • n.trim() - remove espaços no início e no final da string

template literals - string que interage com variáveis diretamente dentro da string.

`O meu nome é ${nome}`

Operadores

Aritméticos

  • n-- ou n++ tem a mesma função que n+=1 e n-=1

Relacionais

  • == - igualdade por valor/valores iguais
  • === - igualdade por tipo/valores idênticos
  • != - diferença por valor/valores diferentes
  • !== - diferença por tipo/valores não idênticos

Lógicos

  • ! - Negação/"NÃO"
  • && - Conjunção/"E"
  • || - Disjunção/"OU"

Operador Ternário

  • ? : - operador que representa a função if/else:
    [teste lógico]?["valor se verdadeiro"]:["valor se falso"]
    ex.: (media > 7) ?"APROVADO":"REPROVADO"
